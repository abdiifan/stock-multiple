How It Works

From Excel to insight in seconds

No server, no login, no setup. PharmaTrack runs entirely in your browser — your data never leaves your machine.

📤
01 / UPLOAD
Drop your SAP extract
Upload any standard SAP inventory export (.xlsx / .xls). Column headers are matched case-insensitively.
🔍
02 / FILTER
Auto-clean the data
Project stock (Q), purely numeric SAP codes, and non-medical material groups are automatically excluded. Only pharma items remain.
📊
03 / ANALYSE
Explore seven views
Navigate between Dashboard, Transit, Expiry, QC, Branch, Flow, and Data Preview. Apply plant and material-group filters per page.
⬇
04 / EXPORT
Download what you need
Every view and drilldown offers one-click CSV and Excel export. Expired items, transit lists, QC registers — ready to share.

What you need to get started

📋
Required Excel Columns
  • Material · Material Description · Plant · Plant Name
  • Storage Location · Description of Storage Location
  • Special Stock Type · Special Stock Type Description
  • Unrestricted Stock · Stock in Quality Inspection · Blocked Stock
  • Stock in Transit · Batch · Inventory Valuation Type
  • Material Group Name · Shelf Life Expiration Date
  • Value of Unrestricted Stock · Value of Stock in Transit · Value in QC
🌐
Browser & Format
  • Any modern browser — Chrome, Firefox, Edge, Safari
  • .xlsx or .xls file from SAP inventory extract
  • Files over 25 MB will show a size warning before parsing
  • No installation, account, or internet connection required after load
  • Reconciliation rules persist across sessions via localStorage
🔒
Data Privacy
  • All processing happens locally in your browser
  • No data is sent to any server at any point
